HTML Tags Reference

A comprehensive guide to all HTML tags and their meanings

Document Structure Tags
<!DOCTYPE>
Defines the document type and HTML version.
<!DOCTYPE html>
<html>
The root element of an HTML page.
<html lang="en">...</html>
<head>
Contains meta-information about the document.
<head>...</head>
<title>
Specifies the title of the document (shown in browser tab).
<title>My Page</title>
<body>
Contains the visible page content.
<body>...</body>
Text Content Tags
<h1> to <h6>
Heading elements from most important (h1) to least important (h6).
<h1>Main Heading</h1>
<p>
Defines a paragraph.
<p>This is a paragraph.</p>
<div>
A generic container for flow content.
<div>Content here</div>
<span>
A generic inline container for phrasing content.
<span>inline content</span>
<br>
Inserts a single line break.
Line one<br>Line two
<hr>
Represents a thematic break between paragraph-level elements.
<p>First section</p><hr><p>Second section</p>
Text Formatting Tags
<strong>
Indicates strong importance (typically displayed as bold).
<strong>Important!</strong>
<em>
Indicates emphasis (typically displayed as italic).
<em>Emphasized text</em>
<b>
Bold text without extra importance.
<b>Bold text</b>
<i>
Italic text without extra emphasis.
<i>Italic text</i>
<u>
Underlined text.
<u>Underlined text</u>
<mark>
Highlighted or marked text.
<mark>Highlighted text</mark>
<small>
Smaller text (typically one size smaller).
<small>Small text</small>
<sub>
Subscript text.
H<sub>2</sub>O
<sup>
Superscript text.
E=mc<sup>2</sup>
List Tags
<ul>
Defines an unordered (bulleted) list.
<ul><li>Item</li></ul>
<ol>
Defines an ordered (numbered) list.
<ol><li>First</li></ol>
<li>
Defines a list item.
<li>List item</li>
<dl>
Defines a description list.
<dl><dt>Term</dt><dd>Definition</dd></dl>
<dt>
Defines a term in a description list.
<dt>HTML</dt>
<dd>
Defines a description in a description list.
<dd>HyperText Markup Language</dd>
Link and Media Tags
<a>
Defines a hyperlink.
<a href="https://example.com">Link</a>
<img>
Embeds an image.
<img src="image.jpg" alt="Description">
<audio>
Embeds sound content.
<audio controls><source src="audio.mp3"></audio>
<video>
Embeds video content.
<video controls><source src="video.mp4"></video>
<iframe>
Embeds another HTML page.
<iframe src="page.html"></iframe>
Table Tags
<table>
Defines a table.
<table>...</table>
<tr>
Defines a table row.
<tr>...</tr>
<th>
Defines a table header cell.
<th>Header</th>
<td>
Defines a table data cell.
<td>Data</td>
<thead>
Groups header content in a table.
<thead>...</thead>
<tbody>
Groups body content in a table.
<tbody>...</tbody>
<tfoot>
Groups footer content in a table.
<tfoot>...</tfoot>
<caption>
Defines a table caption.
<caption>Monthly Sales</caption>
Form Tags
<form>
Defines an HTML form for user input.
<form action="/submit" method="post">...</form>
<input>
Defines an input control.
<input type="text" name="username">
<textarea>
Defines a multi-line text input control.
<textarea rows="4"></textarea>
<button>
Defines a clickable button.
<button type="submit">Submit</button>
<select>
Defines a drop-down list.
<select><option>Option 1</option></select>
<option>
Defines an option in a drop-down list.
<option value="1">First</option>
<optgroup>
Groups related options in a drop-down list.
<optgroup label="Group">...</optgroup>
<label>
Defines a label for an input element.
<label for="name">Name:</label>
<fieldset>
Groups related elements in a form.
<fieldset>...</fieldset>
<legend>
Defines a caption for a fieldset.
<legend>Personal Info</legend>
Semantic Tags (HTML5)
<header>
Represents introductory content or navigational aids.
<header>...</header>
<footer>
Represents the footer of a document or section.
<footer>...</footer>
<nav>
Defines navigation links.
<nav><a href="/">Home</a></nav>
<main>
Specifies the main content of a document.
<main>...</main>
<article>
Defines independent, self-contained content.
<article>...</article>
<section>
Defines a section in a document.
<section>...</section>
<aside>
Defines content aside from the content it is placed in.
<aside>...</aside>
<figure>
Specifies self-contained content like illustrations.
<figure><img src="img.jpg"></figure>
<figcaption>
Defines a caption for a figure element.
<figcaption>Caption</figcaption>
Meta Tags
<meta>
Provides metadata about the HTML document.
<meta charset="UTF-8">
<link>
Defines the relationship between a document and an external resource.
<link rel="stylesheet" href="styles.css">
<style>
Defines style information for a document.
<style>body { color: red; }</style>
<script>
Defines a client-side script.
<script src="script.js"></script>
<noscript>
Defines an alternate content for users without script support.
<noscript>Please enable JavaScript</noscript>
